TradeHive

TradeHive is a trading journal app created using Glide, a low-code application that allows anyone to create apps from spreadsheets. The app features interactive charts, dashboards, learning and resources section, and is designed to be easy to use, customizable, interactive, educational, and secure. The need for the project was to provide a personalized and effective trading journal solution that would help the customer achieve their trading goals. The app uses MS Excel as the data source and integrates with Native Glide charts. The app is designed to be professional and creative. It has a sleek and modern interface that matches the user's branding and preferences.
