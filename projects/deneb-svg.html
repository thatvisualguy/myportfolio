PowerBI Project

Deneb/SVG Visuals

Custom Visuals in Power BI take the data visualization game to a whole new level. Deveb and SVG visuals are at the forefront of this. In one of my recent projects, I had to replicate Deveb and SVG visuals created by Andrzej Leszkiewicz, David Bacci, and Kerry Kolosko - 3 of the prominent creators of custom visuals using Deneb and SVG. The project included forking a copy of Andrzej's Github repo which contained the code of all the custom visuals my client wanted, modifying the code according to the client's color scheme, creating a PBIX file, and sending it to them. The same process was to be followed for David's and Kerry's Visuals.
