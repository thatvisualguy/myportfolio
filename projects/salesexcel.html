Sales Dashboard

The dashboard aimed to provide a comprehensive and interactive view of the manpower, expenditure, and production volume data for the past two years across different manufacturing units. The dashboard allows users to filter and compare the data by various dimensions, such as year, skill level, and expenditure type. The dashboard also provides insights based on data analysis and best practices to optimize workforce allocation, reduce operational costs and increase production efficiency. The dashboard was built using MS Excel and Power Query for data processing and cleaning and MS Excel for data visualization and interactivity. It is deployed on a shared folder for easy access and sharing among the stakeholders.
