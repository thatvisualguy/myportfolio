Power Automate Project

Purchase Order Approvals

This Power Automate project streamlines the Purchase Order (PO) approval process. The user submits a PO requirement through Microsoft Forms, and based on the PO amount, the approval request is automatically routed to the appropriate approver, such as a manager, MD, or supervisor. Once approved, a Microsoft Teams notification is sent, and the relevant Excel file in SharePoint is updated. This automated workflow enhances efficiency, reduces manual intervention, and ensures real-time updates for tracking purchase orders, ensuring the process is smooth and well-documented.
