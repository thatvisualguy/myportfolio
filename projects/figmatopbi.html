PowerBI Project

Figma to Power BI

This Power BI project converted 13 unique Figma dashboard designs into interactive Power BI reports, incorporating third-party visuals such as SVG, Deneb visuals, and app store components. Each dashboard, with two pages, integrated various data sources to faithfully replicate the original designs. The result is a robust set of dashboards offering rich data insights and engaging visual experiences.
